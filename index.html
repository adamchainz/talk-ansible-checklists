Techniques

Ansible Playbooks are checklists

- name: apt package
  apt:
    pkg: nginx

- name: conf file
  template:
    src: etc_nginx_nginx.conf.j2
    dest: /etc/nginx/nginx.conf
    mode: 0644
    owner: root
    group: root

- name: 50x.html
  copy:
    src: 50x.html
    dest: /usr/share/nginx/html/timeout-50x.html
